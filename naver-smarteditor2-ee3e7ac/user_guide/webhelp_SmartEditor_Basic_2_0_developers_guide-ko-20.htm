표 생성과 간단 편집기

SmartEditor Basic 2.0은 표 관련 기능으로 표 생성 기능과 표 편집 기능을 제공한다.

표를 생성할 때에는 행과 열의 개수를 지정할 수 있으며, 테두리 스타일, 테두리 두께, 테두리 색, 셀 배경색을 직접 입력하거나 제공되는 표 스타일 중에서 선택할 수 있다.
